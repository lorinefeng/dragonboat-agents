# Task Prompt

Read `.dragonboat/skills/dragonboat-steerer.md` and `.dragonboat/crew-lessons.md`.

Implement a small fullstack feature: add a sortable task list endpoint and a matching UI control.

First assess Delegation Fit. If crew-fit, draft a crew plan for backend, frontend, and QA rowers. Wait for human confirmation before starting rowers.

Acceptance:

- Backend publishes the endpoint contract.
- Frontend consumes the contract without guessing missing fields.
- QA runs one API check and one UI check.
- Each rower submits mailbox handoff and evidence before claiming done.
